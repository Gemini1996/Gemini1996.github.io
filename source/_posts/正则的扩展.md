## RegExp构造函数
### ES5构造函数
第一种情况，第一个参数是字符串，第二个参数是正则表达式的修饰符
```js
var regex = new RegExp('xyz', 'i');
// 等价于
var regex = /xyz/i;
```
第二种情况，参数是一个正则表达式
```js
var regex = new RegExp(/xyz/i);
// 等价于
var regex = /xyz/i;
```
